📊 ETL Data Pipeline

Projeto de Portfólio desenvolvido com Python para simular um processo completo de ETL (Extract, Transform and Load) aplicado à análise de dados de vendas.

---

🚀 Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQLite
- SQLAlchemy
